Deskripsi Pekerjaan

Kami mencari Junior Android Developer yang passionate dan bersemangat untuk bergabung dengan tim pengembangan mobile kami. Anda akan bertanggung jawab dalam mengembangkan aplikasi Android berkualitas tinggi yang user-friendly dan sesuai dengan standar industri terkini.

Posisi ini sangat cocok untuk fresh graduate atau developer dengan pengalaman 1-2 tahun yang ingin mengembangkan karir di bidang mobile development, khususnya platform Android.

Tanggung Jawab

  • Mengembangkan dan memelihara aplikasi mobile menggunakan Kotlin atau Java
  • Mengimplementasikan desain UI/UX dengan hasil akhir yang responsif dan intuitif
  • Merancang arsitektur aplikasi yang terstruktur, mudah dipelihara, dan mendukung pengembangan tim
  • Melakukan integrasi dengan REST API dan mengelola data menggunakan Firebase atau SQLite
  • Melakukan debugging, testing, dan optimasi performa aplikasi
  • Berkolaborasi dengan tim designer, backend developer, dan project manager
  • Mengikuti perkembangan teknologi Android terbaru dan best practices

Kualifikasi

  • Minimal D3/S1 jurusan Teknik Informatika, Sistem Informasi, atau bidang terkait (Fresh Graduate welcome)
  • Menguasai bahasa pemrograman Kotlin atau Java untuk Android
  • Familiar dengan Android Studio dan development tools lainnya
  • Memahami konsep REST API, JSON parsing, dan integrasi web services
  • Pengalaman dengan Firebase (Authentication, Firestore, Storage) atau SQLite
  • Memahami Material Design Guidelines dan UI/UX principles
  • Mampu bekerja dalam tim dan memiliki komunikasi yang baik
  • Memiliki portfolio aplikasi Android (publish di Play Store menjadi nilai tambah)
